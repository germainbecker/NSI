Consignes¶

L'objectif est d'écrire dans le langage Python les différents algorithmes écrits en langage naturel dans les exercices sur les bases de l'algorithmique.

On reprend les numéros des exercices et on a récrit leurs énoncés ci-dessous. Vous pourrez vérifier les différentes réponses à vos questions en affichant les valeurs des variables.

Aidez-vous si besoin des rappels sur [les bases du langage Python](http://info-mounier.fr/premiere_nsi/bases/bases_python.php).
